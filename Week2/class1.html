테이블

  • 격자로 자료를 표현
  • 셀이라고 불리는 기본 단위를 행과 열로 구분해 표현
  • Html상에는 열에 해당하는 태그가 없음

태그

  • table : 테이블 삽입
  • tr : 표에 행 삽입
  • th : 포에 제목 셀 설정
  • td: 표에 일반 셀 설정 
      <th></th>
  <th>월</th>
  <th>화</th>
  <th>수</th>

      <tr>
      <td>1교시</td>
      <td>영어</td>
      <td>영어</td>
      <td>영어</td>
  </tr>
  <tr>
      <td>2교시</td>
      <td>수학</td>
      <td>수학</td>
      <td>화학</td>
  </tr>
    오른쪽 위 소스코드 보기로 확인 가능

미디어

  • img 태그는 내용불가, audio, video 는 내용 가능

    이미지

  • img 태그
    • 이미지가 없다면 alt에 적은 내용이 나타남

      음악삽입

  • 웹 브라우저마다 모양이 다름
  • 확장자에 문제가 있을 때는 source와 type을 활용 
    <audio controls="controls">
  <source src="Kalimba.mp3", type="audio/mp3/">
  <source src="Kalimba.ogg", type="audio/ogg/">
</audio>

동영상

  • video 태그활용
  • poster 속성을 사용해 로딩중에 이미지를 설정
